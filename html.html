HTML

Esqueleto do site

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a base de quase toda a web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação, usada para estruturar o conteúdo de uma página na internet.

Imagine que você está construindo uma casa: o HTML seria a estrutura básica — as paredes, o telhado, as portas e as janelas. Ele define elementos como títulos, parágrafos, imagens, links, listas e botões, organizando-os de maneira lógica.

Cada elemento HTML é representado por tags (etiquetas), que são interpretadas pelo navegador (como Chrome, Firefox ou Edge) para exibir o conteúdo da forma como vemos. Por exemplo:

  • <h1> para um título principal
  • <p> para um parágrafo
  • <a> para um link
  • <img> para uma imagem

Sozinho, o HTML cria páginas simples e estáticas. Para deixá-las bonitas e interativas, usamos CSS (para estilização) e JavaScript (para funcionalidades). Mas tudo começa com o HTML — ele é o esqueleto essencial de qualquer site.
