CSS

Aparência

Se o HTML é o esqueleto de um site, o CSS é a pele, a roupa e o visual. CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). É a linguagem usada para estilizar e embelezar as páginas web.

Enquanto o HTML cuida da estrutura e do conteúdo (títulos, parágrafos, imagens), o CSS é responsável por toda a parte visual, como:

  • Cores: de fundo, de texto, de bordas.
  • Fontes: tipo, tamanho e peso.
  • Layout: posicionamento dos elementos na página, criação de colunas, espaçamentos.
  • Responsividade: adaptar o site para telas de celular, tablet e desktop.

A magia do CSS é a separação entre conteúdo e formatação. Você pode criar um único arquivo CSS que define o visual de todo o site, e se quiser mudar a cor de todos os títulos, por exemplo, basta alterar em um único lugar.

Sozinho, o CSS não é uma linguagem de programação (como JavaScript), mas é absolutamente essencial para criar experiências agradáveis e visualmente atraentes na web. É a ferramenta que transforma uma simples estrutura de texto em um site bonito e profissional.
